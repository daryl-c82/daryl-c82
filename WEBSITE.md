# Journey Inspired Website

A professional, responsive website for Journey Inspired - a travel-tech initiative focused on AI-enabled travel experiences across health, education, and property tourism.

## Overview

This website showcases Journey Inspired's mission to deliver personalized, holistic-access journeys through digital-first solutions.

## Structure

The website consists of three main files:

- **index.html** - Main HTML structure with semantic sections
- **styles.css** - Responsive CSS styling with modern design
- **script.js** - Interactive JavaScript functionality

## Features

### Design Features
- Modern, professional gradient design
- Fully responsive layout (desktop, tablet, mobile)
- Smooth scroll navigation
- Interactive hover effects
- Animated elements on scroll
- Mobile-friendly hamburger menu

### Content Sections
1. **Hero Section** - Eye-catching introduction with call-to-action
2. **About Section** - Overview of Journey Inspired with key features
3. **Services Section** - Three core verticals:
   - Health & Wellness Travel
   - Education Travel
   - Property Discovery Tourism
4. **Vision Section** - Future plans and technology focus
5. **Contact Section** - Contact information and links
6. **Footer** - Additional navigation and information

## Technologies Used

- HTML5 with semantic markup
- CSS3 with custom properties (CSS variables)
- Vanilla JavaScript (ES6+)
- Responsive design using CSS Grid and Flexbox
- Intersection Observer API for scroll animations

## How to View

1. Open `index.html` in a web browser
2. Or serve using a local web server:
   ```bash
   python3 -m http.server 8080
   ```
   Then navigate to `http://localhost:8080/index.html`

## Browser Compatibility

The website is compatible with all modern browsers including:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Responsive Breakpoints

- Desktop: > 768px
- Tablet: 481px - 768px
- Mobile: ≤ 480px
